Tentang Saya

Perkenalkan, nama saya Daniel Arifan Budialin. Saat ini, saya berkuliah di Universitas Siber Asia dengan jurusan Informatika. Saya adalah pribadi yang penuh rasa penasaran, yang selalu tertarik untuk mencoba hal baru, salah satunya adalah ngoding. Beberapa bahasa pemrograman yang telah saya pelajari antara lain C#, C++, dan Python.

Awal saya terinspirasi untuk mulai ngoding berasal dari hobi saya, seperti membaca komik, menonton film, dan bermain game. Karena kecintaan saya terhadap game, saya pernah mencoba membuat game sendiri menggunakan Unity. Meskipun cukup sulit karena melakukannya seorang diri, saya merasa bangga karena akhirnya saya berhasil menyelesaikan game pertama saya. Pengalaman ini semakin memperkuat minat saya untuk terus belajar dan mengembangkan kemampuan di bidang teknologi dan pemrograman.
